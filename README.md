# 성적 분석 및 성적표 관리 시스템

`vba/GradeAnalysisSystem_CP949.bas`의 주요 흐름을 브라우저 웹앱으로 옮긴 정적 사이트입니다.

## 기능

- NEIS 교과목별일람표 형태의 XLSX, XLS, CSV, TSV 파일 읽기
- 5행 학급, 6행 이후 번호/점수 구조 자동 추출
- 학생명렬 입력과 누락 확인
- 2015 개정 9등급, 2022 개정 5등급 누적퍼센트 기준 분석
- 석차, 중간석차, 백분율, 석차등급, 동석차수 계산
- 학급 통계, 등급 분포, 등급컷, 점수 히스토그램 표시
- 학급/번호 기반 학생 개별 조회
- 현재 학생 또는 전체 학생 성적표 브라우저 PDF 출력
- 분석 결과 CSV 저장

## 사용 방법

1. `index.html`을 브라우저에서 엽니다.
2. `성적자료 불러오기`로 NEIS XLS data 파일을 선택합니다.
3. 생성된 학생명렬에 이름을 입력하거나 엑셀 표를 붙여넣습니다.
4. 교육과정과 누적퍼센트 기준을 확인한 뒤 `분석 시작`을 누릅니다.
5. 학생조회에서 개별 성적표를 확인하고 PDF로 저장합니다.

파일과 성적 데이터는 브라우저 안에서 처리되며 서버로 업로드되지 않습니다.

## 검증

```bash
node --check app.js
npm.cmd run check
```
